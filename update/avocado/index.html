It looks like you have Avocado installed. Avocado installs a version of Origami with incorrect permissions that may cause your installation to fail.



If you see an installation error, follow these instructions:


  1. Switch to the Finder (the leftmost icon in the Dock)
  2. Open the Go menu
  3. While holding the option key on your keyboard, select Library from the menu
  4. Find the Graphics folder and drag it onto the Desktop
  5. Re-run the Origami installer
  6. Copy any files you want to keep from the old Graphics folder to the new Graphics folder


If you're still having trouble, help is available in the Origami Community Facebook group.
