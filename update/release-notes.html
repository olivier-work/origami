Version 2.2

  • New patches for working with live data, styling rich text, and more. Easily download JSON, work with a REST api, or download other web content. Then integrate it into your prototypes with a suite of advanced structure patches.
  • Added a new JSON Importer patch for parsing JSON data from the web.
  • Added a new REST Request patch for working with REST services.
  • Added a new Image at URL patch for asynchronously downloading and caching images.
  • Added a new String at URL patch for asynchronously downloading text content.
  • Added new patches for styling text: String Formatter and Image with Formatted Strings.
  • Added a new AppleScript for executing AppleScript files.
  • Added Sound Player Pro, a more flexible sound player patch.
  • Added new patches for working with complex structures: Structure Path Member, Structure Multiple Path Members, Structure All Keys, Structure All Values, and Structure Shuffle.
  • Added some helpful utility patches: Description, File Updated, Log to Console, and FPS.

Version 1.3

  • Added a new Inline Values feature which lets you see and modify live-updating values for every input port in the Quartz Composer editor
  • Added a new Pop Animation patch, which animates a changing value with a spring. You can use the same Bounciness and Speed values on iOS and OS X using the Pop animation library
  • Added a new Live Image patch, which imports an image from a location on your local machine and automatically updates its image when the file changes (and optionally saving a copy of the image if the file moves)
  • Added a new Delay patch, which lets you delay any value for a specified duration in seconds
  • Added a new Drag and Drop patch, which lets you get drag and drop events on the viewer window and is useful for prototyping desktop UIs with file dragging from your Mac on to your interface
  • Added a new Repeating Pulse patch, which outputs a repeated pulse separated by a duration in seconds—useful for getting a transition to repeat without needing to interact with the composition
  • The Scroll patch now supports the Magic Mouse and trackpad
  • Added a new Mouse Scroll patch, which gives you scroll data from the Magic Mouse and trackpad
  • Added a new Multi-stop Progress patch, which gives you the progress of a value inside of a structure of stops
  • Added new window management options (Resize to Thirds, Picture in Picture and Multi-monitor Layout) to the Window menu
  • Now when you delete any patch, Quartz Composer will attempt to retain connections between the top ports
  • Holding command and option while dragging in an image into the editor will add a Live Image patch
  • The Browser Chrome patch will now restore the status bar visibility to its previous state after the composition is stopped
  • The Counter 2 patch now supports iterators and will now also automatically set its value to the “Jump to Number” value when the composition starts
  • The Cursor Control patch will now allow for changing the cursor while the mouse is down
  • Fixed a bug in Quartz Composer where interaction ports could sometimes have their last port name clipped
  • The Value to Boolean patch now evaluates Cocoa objects as false
  • Fixed a divide by zero issue with the Progress patch
  • Fixed a NaN output case in the Counter 2 patch
